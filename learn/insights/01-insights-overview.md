# 1. Insights Overview

> Insights is where you review performance instead of entering daily transactions.

## What Insights Is For

Use Insights to:

- Review closed monthly snapshots
- Run financial reports
- Compare performance across periods
- Drill into the numbers behind summaries
- Read executive commentary based on actual figures

## What Insights Is Not For

Insights is not where you create invoices, record payments, or enter expenses.

Use:

- Billing for quotes, invoices, payments, statements, and credit notes
- Finance for income, expenses, categories, and distributions
- Accounting for journals, ledgers, trial balance, periods, and exports

## Good Questions To Ask

| Question | Where To Look |
|----------|---------------|
| Which months are closed? | Snapshots |
| Was a month profitable? | Snapshot detail |
| What changed month over month? | Reports and commentary |
| Which division is strongest? | Division report |
| What transactions make up this number? | Drilldown panel |
