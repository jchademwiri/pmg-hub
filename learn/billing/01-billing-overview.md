# 1. Billing Overview

> Billing is where money starts its journey into the system.

## What Billing Is For

Use Billing to:

- Send quotes to potential clients
- Issue invoices for approved or completed work
- Record payments from clients
- Create credit notes when an invoice needs to be reduced
- View client accounts and statements

## How Billing Connects To The Business

```text
Quote -> Invoice -> Payment -> Income -> Accounting -> Reports
```

Most users do not need to create accounting entries manually. Issued invoices and recorded payments create the financial activity behind the scenes.

## Divisions

Every quote and invoice belongs to a division.

| Acronym | Division |
|---------|----------|
| AWS | Apex Web Solutions |
| TES | Tender Edge Solutions |
| PMG | Playhouse Media Group |

The division affects document numbers, logos, reporting, income, and division performance.

## Daily Billing Checklist

1. Confirm the client exists.
2. Choose the correct division.
3. Create a quote if the client has not approved the work.
4. Create or issue an invoice when it is time to bill.
5. Record payment only after money arrives.
6. Check the client statement if there is confusion about what is owed.
