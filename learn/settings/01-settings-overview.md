# 1. Settings Overview

> Settings is the admin area for system configuration.

## What Settings Controls

Settings can affect:

- Organisation profile
- Logos and addresses
- Billing setup by division
- Users and invitations
- Security information
- Data exports
- Database backups and restore

## Who Should Use Settings

Most users can read Settings if they need context, but only admins should change it.

Bad settings can affect invoices, quote PDFs, emails, backups, and access.

## Quick Reference

| I want to... | Go to... |
|-------------|----------|
| Update company details | Settings -> Organisation |
| Configure division billing | Settings -> Billing |
| Invite a user | Settings -> Users |
| Review access information | Settings -> Security |
| Export or back up data | Settings -> Data |
