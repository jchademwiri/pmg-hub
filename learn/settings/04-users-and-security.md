# 4. Users And Security

> Users and security settings help control who can access the system.

## Inviting Users

1. Go to Settings -> Users.
2. Click Invite User.
3. Enter the correct email address.
4. Send the invite.
5. Ask the user to sign in and confirm access.

## Good Access Habits

- Each person should have their own account.
- Do not share logins.
- Remove access when someone no longer needs the system.
- Keep admin access limited.
- Review pending invitations.

## Security Page

The Security page explains the current access-control setup and future controls.

Use it to understand:

- Who should have admin access
- Which controls are active
- Which controls are informational or planned

## Common Mistakes

| Mistake | Better Approach |
|---------|-----------------|
| Sharing one login | Invite each user separately |
| Giving everyone admin access | Give admin access only when needed |
| Leaving old users active | Remove or disable unused access |
